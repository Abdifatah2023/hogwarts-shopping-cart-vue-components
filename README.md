# Hogwarts Shopping Cart (Vue Components)

This project is a Vue 3 + Vite template designed to help you get started building magical shopping cart components.

---

## 🛠 Project Setup

To get your cauldron bubbling and install all necessary dependencies, run:

```sh
npm install

```

### Compile and Hot-Reload for Development

Use this command to start the local development server:

```sh
npm run dev

```

### Compile and Minify for Production

When you're ready to ship your components to the wizarding world:

```sh
npm run build

```
